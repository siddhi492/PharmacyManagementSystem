# 💊 Pharmacy Management System

A simple, robust web-based application for managing pharmacy operations including medicines, users, orders, and sales. This Java-based project is built using Spring MVC, JSP/Servlets, and MySQL, following the MVC architecture pattern.

🚀 Features
🔐 User Authentication

Login and registration for new and existing users

Logout functionality

💊 Medicine Management

Add and list medicines

Track available medicine stock

📦 Order Management

Place and manage customer orders

Link orders to specific users and medicines

📈 Sales Tracking

Record completed sales

View sales history and reports

👨‍⚕️ Admin Module

Centralized control of pharmacy operations

🛠️ Tech Stack
Frontend: HTML, CSS, JSP

Backend: Java, Servlets, Spring MVC

Database: MySQL

Build Tool: Maven

Server: Apache Tomcat
